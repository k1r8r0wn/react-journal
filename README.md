# React Journal

### Simple journal with posts using React.js library.

Highlights:

- Independent components structure.
- [Airbnb JavaScript Style Guide used](https://airbnb.io/javascript/react/)

#### Steps to run:

1. Simply run:
```bash
npm install
npm start
```
2. Go to `localhost:3000` in your browser.

Tada!
